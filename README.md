## Getting Started

### Installation 
```bash
 npm install
```

### Running the app
```bash
 npm run dev or yarn dev
```

Runs the app in the development mode. 
Open http://localhost:3000 to view it in your browser.

## Smart contract

### Compilation
```bash
 npx hardhat compile
```

### Test
```bash
 npx hardhat test
```

### Deploy
```bash
 npx hardhat run scripts/deploy.js --network kovan
```
